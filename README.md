# The Elite Opener™

The AI Conversation Intelligence System for Real Estate Agents.

## Files

- `index.html` — Sales page (your public-facing landing page)
- `app.html` — The product tool (what customers access after purchase)
- `vercel.json` — Routing config for Vercel deployment

## Deploy to Vercel (2 minutes)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and click **New Project**
3. Import your GitHub repo
4. Framework: **Other** (no build step needed)
5. Click **Deploy**

Done. Your sales page is live at `yourdomain.com` and the tool is at `yourdomain.com/app`.

## Custom Domain

In Vercel dashboard: Settings → Domains → Add your domain.
Point your DNS to Vercel's nameservers.
